simFlix


This web application will provide users with access to information about different movies, directors, genres, ratings and reviews. Users will be able to sign up, update their personal information, and create a list of their favorite movies. User will be able to rate movies and share their opinions as well.

Request URL HTTP Method Response
Get a list of movies /Movies GET Returns a JSON list of movies.
Get data about the movie by title /movies/:title GET Returns a JSON data about a specific movie by title from parameter.
Get data about the director by director name from parameter /directors/:directorName GET Returns a JSON data about director.
Add new user /users/:newUser POST Add new user to JSON object.
Allow users to update their user information /users/:Username PUT Return the data fields updated by the user.
Delete the user /users/:deleteUser DELETE Return a text confirming that the user was deleted.
Add new movie to list of favorite /favorite/:movieName POST Return a text confirming taht the given movie was added successfully.
